# aleclv GitHub Profile Explorer

A premium GitHub profile explorer dashboard built with **HTML, CSS and vanilla JavaScript**.

Search any GitHub username and explore repositories, languages, and activity through a clean, recruiter-ready interface.

## Features

- GitHub user search
- Repository explorer
- Highlighted repositories based on scoring
- Language filtering
- Repository keyword search
- Sorting by stars or recent updates
- Shareable URLs (`?user=username`)
- Skeleton loading UI
- Dark / light mode
- EN / ES language toggle

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- GitHub REST API
